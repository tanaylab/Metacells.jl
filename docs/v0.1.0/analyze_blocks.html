Metacells.AnalyzeBlocks.compute_blocks_genes_linear_fractions! — Function 
function compute_blocks_genes_linear_fractions!(
    daf::DafWriter;
    overwrite::Bool = ```false```,
)::Nothing

An estimated linear fraction of the UMIs of each gene in each block. This is just the total UMIs of the gene in the metacell divided by the total UMIs of the metacell, which is the "best" estimate assuming multinomial sampling noise. However, this is sensitive to a few metacells with very high expression levels ("bursty" genes).

Inputs

Axes

gene (required): Sequenced genes.

block (required): Distinct groups of metacells with "very close" cell state.

Vectors

gene @ is_excluded ::Bool (required): A mask of genes that are excluded from consideration.

Matrices

gene, block @ UMIs ::Union{UInt16, UInt32, UInt64, UInt8} (required): The total number of UMIs of each gene in each block.

Outputs

Matrices

gene, block @ linear_fraction ::Union{Float32, Float64} (guaranteed): The linear fraction of the UMIs of each non-excluded gene out of the total UMIs in each block.
