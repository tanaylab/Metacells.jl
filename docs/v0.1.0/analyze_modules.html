Compute and set vector_of_mean_euclidean_modules_cells_distance_per_metacell and and set vector_of_std_euclidean_modules_cells_distance_per_metacell .

Inputs

Axes

block (required): Distinct groups of metacells with "very close" cell state.

metacell (required): Minimal-sized groups of cells for robust point estimates.

cell (required): Sequenced single cells.

gene (required): Sequenced genes.

module (required): Groups of non-lateral genes that together predict the rest of the genes at a specific region of the manifold.

Vectors

cell @ metacell ::AbstractString (required): The unique metacell each cell belongs to.

metacell @ block ::AbstractString (required): The unique block each metacell belongs to.

cell @ total_UMIs ::Union{UInt16, UInt32, UInt64, UInt8} (required): The total number of UMIs of all the non-excluded genes in each cell.

Matrices

block, gene @ module ::AbstractString (required): The module each gene belongs to in each block.

gene, cell @ UMIs ::Union{UInt16, UInt32, UInt64, UInt8} (required): The number of UMIs collected for each gene for each cell.

Outputs

Vectors

metacell @ mean euclidean modules cells distance ::Union{Float32, Float64} (created): In each metacell, the mean of the euclidean distances of between its cells and the metacell's linear fraction of the modules of the metacell's block's neighborhood.

metacell @ std euclidean modules cells distance ::Union{Float32, Float64} (created): In each metacell, the standard deviation of the euclidean distances of between its cells and the metacell's linear fraction of the modules of the metacell's block's neighborhood.
