Metacells.AnalyzeModules.compute_blocks_modules_is_used! — Function 
function compute_blocks_modules_is_used!(
    daf::DafWriter;
    min_used_module_covered_UMIs_fraction::Real = ```0.67```,
    overwrite::Bool = ```false```,
)::Nothing

Whether each gene module is used by each block. We only use modules that have a high fraction of covered gene UMIs (specifically, at least min_used_module_covered_UMIs_fraction ).

Inputs

Axes

block (required): Distinct groups of metacells with "very close" estimated cell state.

module (required): A local (per block) gene module.

Matrices

block, module @ covered_UMIs ::Union{UInt16, UInt32, UInt64, UInt8} (required): The total UMIs of the covered genes in each gene module in each block.

block, module @ total_UMIs ::Union{UInt16, UInt32, UInt64, UInt8} (required): The total UMIs of the genes in each gene module in each block.

Outputs

Matrices
