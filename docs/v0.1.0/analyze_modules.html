Metacells.AnalyzeModules.compute_blocks_modules_is_strong! — Function 
function compute_blocks_modules_is_strong!(
    daf::DafWriter;
    min_downsamples::Integer = ```750```,
    min_downsamples_quantile::AbstractFloat = ```0.05```,
    max_downsamples_quantile::AbstractFloat = ```0.5```,
    min_strong_UMIs::Integer = 8,
    min_strong_cells::Integer = ```12```,
    overwrite::Bool = ```false```,
)::Nothing

A mask of the strong modules that have enough UMIs in enough cells. Enough UMIs means that, if we downsample the cells to a common total covered UMIs using downsamples with the min_downsamples , min_downsamples_quantile , max_downsamples_quantile , the expected total UMIs of the module will be at least min_strong_UMIs . Enough cells mean that this will happen in at least min_strong_cells in the environment. Modules which achieve this are marked as strong enough to be meaningfully applicable to single cells.

Inputs

Axes

cell (required): Sequenced single cells.

metacell (required): Minimal-sized groups of cells for robust point estimates.

block (required): Distinct groups of metacells with "very close" cell state.

gene (required): Sequenced genes.

module (required): Groups of genes that together predict the rest of the covered genes.

Vectors

cell @ metacell ::AbstractString (required): The unique metacell each cell belongs to.

metacell @ block ::AbstractString (required): The unique block each metacell belongs to.

cell @ covered_UMIs ::Union{UInt16, UInt32, UInt64, UInt8} (required): The total number of UMIs of all the covered genes in each cell.

Matrices

block, block @ is in environment ::Bool (required): For each block, the (column) mask of nearby blocks in its expanded environment.

gene, cell @ UMIs ::Union{UInt16, UInt32, UInt64, UInt8} (required): The number of UMIs collected for each gene for each cell.

block, gene @ module ::AbstractString (required): The module each gene belongs to in each block.

block, module @ is_found ::Bool (required): A mask of the modules that were found for each block.

Outputs

Matrices

block, module @ is_strong ::Bool (guaranteed): A mask of the modules that have enough UMIs in enough cells.
