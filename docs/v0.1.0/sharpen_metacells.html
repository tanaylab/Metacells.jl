Sharpen Metacells

Metacells.SharpenMetacells — Module

Compute "better" metacells based on the local linear model approximating the manifold.

Metacells.SharpenMetacells.sharpen_metacells! — Function 
function sharpen_metacells!(
    original::DafReader,
    sharpened::DafWriter;
    scale::Bool = ```false```,
    migrate::Bool = ```true```,
    min_downsamples::Integer = ```750```,
    min_downsamples_quantile::AbstractFloat = ```0.05```,
    max_downsamples_quantile::AbstractFloat = ```0.5```,
    min_cells_in_metacell::Integer = ```12```,
    min_migrated_cells_fraction::AbstractFloat = ```0.01```,
    kmeans_rounds::Integer = ```10```,
    rng::AbstractRNG = default_rng(),
    overwrite::Bool = ```false```,
)::Nothing

Given an original repository with a local linear model approximating the manifold, compute new metacells into the sharpened repository, which hopefully more faithfully capture this manifold.

Original Metacells

Inputs

Axes

cell (required): Sequenced single cells.

metacell (required): Minimal-sized groups of cells for robust point estimates.

block (required): Distinct groups of metacells with "very close" estimated cell state.

gene (required): Sequenced genes.

module (required): A local (per block) gene module.

Vectors

cell @ metacell ::AbstractString (required): The unique metacell each cell belongs to.

metacell @ block ::AbstractString (required): The unique block each metacell belongs to.

gene @ is_covered ::Bool (required): A mask of genes that are covered by the local linear program.

Matrices

gene, cell @ UMIs ::Union{UInt16, UInt32, UInt64, UInt8} (required): The number of UMIs collected for each gene for each cell.

block, gene @ module_index ::Union{UInt16, UInt32, UInt64, UInt8} (required): The index of the gene module each gene belongs to in the environment of each block.

block, module @ is_used ::Bool (required): Whether each gene module is used in the linear model of each block.

block, block @ is in neighborhood ::Bool (optional): For each block, the (column) mask of nearby blocks in its immediate neighborhood.
