function compute_vector_of_is_marker_per_gene!(
    daf::DafWriter;
    min_marker_gene_max_fraction::AbstractFloat = ```0.0001```,
    min_marker_gene_range_fold::Real = ```2```,
    min_marker_quantile::Real = ```0```,
    overwrite::Bool = ```false```,
)::Nothing

Compute and set vector_of_is_marker_per_gene . Such genes:

  • Have a maximal gene expression level of at least min_marker_gene_max_fraction , and
  • Have a range of expression of at least min_marker_gene_range_fold between the maximal and minimal (log base 2) of the expression level. For the minimal expression, we take the min_marker_quantile of the expression levels in the population.

Inputs

Axes

gene (required): Sequenced genes.

metacell (required): Minimal-sized groups of cells for robust point estimates.

Matrices

gene, metacell @ log linear fraction ::Union{Float32, Float64} (required): The log base 2 of the linear fraction of the UMIs of each non-excluded gene in each metacell, out of the total UMIs.

Outputs

Vectors

gene @ is_marker ::Bool (created): A mask of genes that distinguish between cell states.

Metacells.AnalyzeGenes.compute_vector_of_marker_rank_per_gene! — Function 
function compute_vector_of_marker_rank_per_gene!(
    daf::DafWriter;
    overwrite::Bool = ```false```,
)::Nothing

Compute and set vector_of_marker_rank_per_gene . To compute this:

  1. Non-marker genes are given a rank of typemax(UInt32) regardless of their expression level in the metacells (see vector_of_is_marker_per_gene ).
  2. Compute the median of the (log base 2) of the gene expression level for each marker gene across all the metacells.
  3. Compute the per-marker-per-metacell fold factor (absolute difference of the log expression from the median).
  4. Invoke rank_variables to rank the marker genes from most to least significant.

Non-marker genes are given a rank of typemax(UInt32) regardless of their expression level in the metacells.

Inputs

Axes

gene (required): Sequenced genes.

metacell (required): Minimal-sized groups of cells for robust point estimates.

Vectors

gene @ is_marker ::Bool (required): A mask of genes that distinguish between cell states.

Matrices

gene, metacell @ log linear fraction ::Union{Float32, Float64} (required): The log base 2 of the linear fraction of the UMIs of each non-excluded gene in each metacell, out of the total UMIs.

Outputs

Vectors

gene @ marker_rank ::Union{UInt16, UInt32, UInt64, UInt8} (created): The ralative ranks of the marker genes.

Metacells.AnalyzeGenes.rank_variables — Function 
rank_variables(score_per_variable_per_observation::AbstractMatrix{<:AbstractFloat})::AbstractVector{<:Integer}

Given some score_per_variable_per_observation matrix, return a vector of the rank of each variable such that the "most significant" variables are first.

  1. Rank the variables for each observation (1 having the highest score).
  2. For each variable, give it a priority which is a tuple of (1) the minimal rank it has in all observations (2) the maximal score it has in observations where it has that rank (negated).
  3. Sort the variables according to this priority.

This heuristic is useful for focusing on the "most significant" variables in a data set. It is used by compute_vector_of_marker_rank_per_gene! .
