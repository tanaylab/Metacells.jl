Programs

Metacells.Programs — Module

Given a set of raw metacells, partition them into blocks such that all metacells in the same block are within some (fold factor) radius of each other. The centroids of these blocks can serve as a representation of the cell state manifold which is less sensitive to oversampling of common cell states. Group these blocks in overlapping neighborhoods of "similar" blocks for further analysis.
